A curated list of tools for building websites, web apps, mobile apps, and supporting assets. Grouped by purpose rather than popularity.

## AI Assistants and Coding Help

- **Claude (Anthropic)**  
  https://www.anthropic.com  
  Claude Code is particularly useful for reasoning, refactoring, and even non code tasks like organizing files.

- **ChatGPT**  
  https://chat.openai.com  
  Used for brainstorming, structuring ideas, and occasionally drafting PRDs.

## App and Website Builders

- **Lovable**  
  https://lovable.dev  
  Rapidly spin up full stack websites and web apps with an integrated backend and AI gateway. Supabase based infrastructure, abstracted.

- **Google AI Studio**  
  https://aistudio.google.com  
  Ideal for fast prototypes using Gemini APIs. Publishing and deployment options are still limited.

- **Bolt.new**  
  https://bolt.new  
  Suitable for more complex projects where terminal access is required.

- **YouWare AI**  
  https://youware.ai  
  Website and web app builder with model selection and integrated backend through YouBase.

- **Replit**  
  https://replit.com  
  Useful for mid sized projects that sit between prototyping and production.

- **Anything**  
  https://anything.dev  
  Website, web app, and mobile app builder with backend included and simplified App Store publishing.

- **Cursor**  
  https://cursor.sh  
  AI first code editor that works for almost any type of software project.

## Mobile and Mini App Tools

- **Rork**  
  https://rork.app  

- **VibeCode**  
  https://vibecode.app  

- **Natively**  
  https://natively.dev  

- **Wabi**  
  https://wabi.app  
  Focused on building small, contained mini apps.

## Backend, Databases, and Infrastructure

- **GitHub**  
  https://github.com  
  Version control and the central link between code and deployment providers.

- **Netlify**  
  https://www.netlify.com  
  Deployment and hosting with seamless GitHub integration and automatic redeploys.

- **Convex**  
  https://convex.dev  
  Backend platform with real time data and serverless functions.

- **Supabase**  
  https://supabase.com  
  Database, authentication, and backend services.

## Domains, Fonts, and Design Resources

- **Namecheap**  
  https://www.namecheap.com  
  Domain registration and management.

- **Google Fonts**  
  https://fonts.google.com  
  Free, high quality web fonts. Always change the default font.

- **Coolors**  
  https://coolors.co  
  Color palette generator.

- **Mobbin**  
  https://mobbin.com  
  Real world UI inspiration from production apps.

- **Dribbble**  
  https://dribbble.com  
  Conceptual UI and visual design inspiration.

## Assets and Visuals

- **NanoBanana**  
  https://nanobanana.ai  
  Professional assets such as images and icons.

- **Canva**  
  https://www.canva.com  
  Magic Erase, background removal, favicon creation, and Open Graph images.
